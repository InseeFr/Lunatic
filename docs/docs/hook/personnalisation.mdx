# Personnalisation

Par défaut, les composants proposés par `@inseefr/lunatic` sont assez simples avec un style minimal. Vous pouvez personnaliser les champs avec votre propre CSS, mais pour des cas plus complexes, vous pouvez remplacer les composants de bases par vos propres composants à l'aide de l'option `custom`.

```jsx
const custom = {
  Input: MyCustomInput,
  InputNumber: MyCustomInputNumber,
};

function App({ source, data }) {
  const {} = useLunatic(source, data, { custom });
  // ...
}
```
