# Démarrage

Lunatic est une bibliothèque **React** pour afficher des questionnaires conçus sur l'outil [Pogues](https://github.com/inseeFr/pogues) et générés au format [Lunatic-Model](https://github.com/InseeFr/Lunatic-Model) via [Eno](https://github.com/inseeFr/eno).

## Pour commencer

Pour commencer à utiliser Lunatic dans votre application il faut télécharger la dépendance depuis `npm`.

```bash npm2yarn
npm install @inseefr/lunatic
```

Ensuite la librairie se décompose en 2 parties :
- un hook `useLunatic()` qui permet de lire un fichier [Lunatic-Model](https://github.com/InseeFr/Lunatic-Model) et d'offrir les différentes fonctionnalités permettant de naviguer dans les composants.
- un composant `<LunaticComponents/>` qui permet d'afficher la liste des composants.

## Exemple

import source from '../../src/stories/input/source.json';
import {FormExample} from '@site/src/components/FormExample/FormExample';

<FormExample source={source} />
