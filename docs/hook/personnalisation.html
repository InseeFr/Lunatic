Personnalisation

Par défaut, les composants proposés par @inseefr/lunatic sont assez simples avec un style minimal. Vous pouvez personnaliser les champs avec votre propre CSS, mais pour des cas plus complexes, vous pouvez remplacer les composants de bases par vos propres composants à l'aide de l'option custom.

const custom = {
  Input: MyCustomInput,
  InputNumber: MyCustomInputNumber,
};

function App({ source, data }) {
  const {} = useLunatic(source, data, { custom });
  // ...
}

Liste des composants personnalisables​

N'hésitez pas à regarder le type des propriétés des composants originaux.

Base​

  • Button, Composant bouton.
  • Fieldset
  • Label
  • Declaration
  • Declarations

Élément de formulaire​

  • ComponentSet
  • Input, Champs texte avec label.
  • Textarea, Champs texte multiligne avec label.
  • Datepicker
  • Dropdown, Menu déroulant simple.
  • InputNumber, Champs numérique.
  • Modal, Boîte modale.
  • Switch.

Cases à cocher​

  • CheckboxBoolean, Case à cocher unique.
  • CheckboxGroup, Cases à cocher (chaque case correspondant à une réponse).
  • CheckboxOne, Cases à cocher (simule un radio décochable).
  • CheckboxOption, Sous composant représentant une case à cocher.
  • Radio, Boutons radios.
  • RadioOption, Boutons radios.

Menu déroulant​

  • Combobox, élément contenant le champ et le label.
  • ComboboxContainer, élément entourant le champ.
  • ComboboxContentBox, élément entourant le champ.
  • ComboBoxOption, élément dans la liste déroulante
  • ComboboxPanelContainer, élément représentant le menu déroulant.
  • ComboxClearButton, bouton permettant de vider la sélection (croix par défaut).
  • ComboboxLabelSelection, label représentant la valeur sélectionnée.
  • ComboboxInput, champs permettant la recherche dans le menu déroulant.

Suggester​

  • Suggester
  • SuggesterNotifcation
  • Notification

Tableau​

  • Table
  • Tbody
  • Td
  • Th
  • Thead
  • Tr

Sequence​

  • Sequence
  • Subsequence

Rond point​

  • Roundabout
  • RoundaboutItTitle
  • RoundaboutContainer, élément entourant le rond-point.
  • RoundaboutItButton, bouton associé à un élément du rond-point.
  • RoundaboutItContainer
  • RoundaboutLabel
  • RoundaboutPending

Sommaire​

  • SummaryResponses
  • SummaryTitle

Autres​

  • FilterDescription
  • QuestionContext
  • QuestionInformation

Composants racines​

Ces composants sont assez haut niveau et contiennent souvent beaucoup de logique que vous devrez dupliquer. Aussi, ne les personnalisez qu'en dernier recours.

  • LunaticRadioGroup
  • LunaticDatepicker
